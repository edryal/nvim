## Prerequisites

The following tools must be available in your PATH:

> **git** <br>
> **unzip** <br>
> **make** <br>
> **fd** <br>
> **ripgrep** <br>
> **bat** <br>
> **fzf** <br>
> **gcc** <br>
> **java** <br>
> **nodejs** <br>

Optional:

> **golang** <br>
> **lazydocker** <br>

## Java Requirements

### Language Server Configuration
This configuration uses JDTLS (Java Language Server) which requires:
- **Java 21+** to run the language server itself
